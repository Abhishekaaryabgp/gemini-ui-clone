# 📁 Assets Folder

This folder contains all static resources used in the Gemini UI Clone project.

---

## 🖼️ Images & Icons

* `Abhishek.jpeg` – Profile image used in UI
* `Gemini Dotted icon.svg` – Decorative Gemini icon
* `gemini-color.svg` – Colored Gemini logo
* `gemini-text.png` – Gemini text logo
* `GeminiSvg.svg` – Main Gemini SVG icon
* `Nano-Banana.png` – UI illustration asset
* `PRO LOGO.png` – Pro badge/logo
* `profile.png` – User profile image

---

## 📌 Notes

* All assets are used for UI design and layout
* Optimized for web usage
* Stored locally for faster loading

---

## ⚠️ Usage

These assets are used in the project via relative paths:

```html
<img src="assets/profile.png" />
```
